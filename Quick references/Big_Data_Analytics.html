Big Data Analytics - Quick Reference

ConceptDescription
Definition
Process of examining large and varied data sets to uncover hidden patterns, market trends, customer preferences, and other useful business information.
The 5 V's
Volume (amount), Velocity (speed), Variety (types), Veracity (quality), Value (benefit)
Characteristics
Large scale, Distributed processing, Real-time analytics, Heterogeneous data sources
Big Data TechnologiesDescription
Hadoop
Open-source framework for distributed storage and processing of large datasets across clusters of computers.
Spark
Fast and general engine for large-scale data processing with in-memory computing capabilities.
NoSQL Databases
Non-relational databases for unstructured data (MongoDB, Cassandra, HBase, Couchbase)
Kafka
Distributed streaming platform for building real-time data pipelines and streaming applications.
Data Storage SolutionsDescription
HDFS
Hadoop Distributed File System for storing large files across multiple machines.
Data Lake
Storage repository that holds vast amounts of raw data in its native format.
Data Warehouse
System for reporting and data analysis, containing integrated data from multiple sources.
Cloud Storage
Amazon S3, Google Cloud Storage, Azure Blob Storage for scalable storage solutions.
Processing FrameworksDescription
MapReduce
Programming model for processing large data sets with a parallel, distributed algorithm on a cluster.
YARN
Resource management layer in Hadoop for job scheduling and cluster resource management.
Storm
Real-time computation system for processing streaming data.
Flink
Stream processing framework for distributed, high-performance data analysis.
Analytics TypesDescription
Descriptive
What happened? Analyzes past data to understand historical patterns and performance.
Predictive
What might happen? Uses statistical models and ML to predict future outcomes.
Prescriptive
What should be done? Provides recommendations for optimal decisions.
Machine Learning in Big DataDescription
Algorithms
Linear regression, Random forests, Neural networks, Clustering, Association rules
Tools
MLlib (Spark), H2O, Mahout, R, Python (scikit-learn, TensorFlow)
Applications
Customer segmentation, Fraud detection, Recommendation systems, Predictive maintenance
Visualization ToolsDescription
Tableau
Business intelligence and data visualization platform for creating interactive dashboards.
Power BI
Microsoft's business analytics tool for data visualization and business intelligence.
Apache Superset
Open-source data visualization and exploration platform.
D3.js
JavaScript library for creating dynamic, interactive data visualizations in web browsers.
ApplicationsDescription
Healthcare
Predictive analytics for patient outcomes, drug discovery, personalized medicine.
Finance
Fraud detection, risk management, algorithmic trading, credit scoring.
Retail
Customer behavior analysis, inventory optimization, personalized recommendations.
IoT
Sensor data analysis, predictive maintenance, smart city applications.
