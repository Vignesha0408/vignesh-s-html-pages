Information Theory - Quick Reference

ConceptDescription
Definition
Mathematical study of information, communication, and data transmission. Deals with quantification, storage, and communication of information.
Founder
Claude Shannon (1948) - "A Mathematical Theory of Communication"
Applications
Data compression, error correction, cryptography, machine learning, communication systems
Information MeasuresDescription
Entropy
Measure of uncertainty or randomness in a system. H(X) = -Σ p(x) log p(x)
Joint Entropy
Entropy of two variables together. H(X,Y) = -Σ Σ p(x,y) log p(x,y)
Conditional Entropy
Uncertainty of X given Y. H(X|Y) = -Σ Σ p(x,y) log p(x|y)
Mutual Information
Amount of information one variable contains about another. I(X;Y) = H(X) - H(X|Y)
Data CompressionDescription
Lossless
Data can be perfectly reconstructed (Huffman coding, Lempel-Ziv)
Lossy
Some data lost for higher compression (JPEG, MP3)
Shannon's Source Coding Theorem
Data can be compressed to entropy limit without loss
Channel CapacityDescription
Definition
Maximum rate at which information can be transmitted over a communication channel
Shannon-Hartley Theorem
C = B log₂(1 + S/N) where B is bandwidth, S/N is signal-to-noise ratio
Noisy Channel Coding Theorem
Information can be transmitted reliably up to channel capacity with error correction
Error CorrectionDescription
Hamming Code
Linear error-correcting code that can detect and correct single-bit errors
Reed-Solomon
Non-binary cyclic code used in CDs, DVDs, QR codes
LDPC
Low-density parity-check codes approaching Shannon limit
Turbo Codes
Approach Shannon limit, used in mobile communications
Key TheoremsDescription
Shannon's Theorems
Source coding theorem, noisy channel coding theorem, rate-distortion theory
Data Processing Inequality
I(X;Y) ≥ I(X;Z) if X→Y→Z forms a Markov chain
Fano's Inequality
Relates probability of error to conditional entropy
ApplicationsDescription
Communication
Optimizing data transmission, error correction, modulation schemes
Data Storage
Compression algorithms, error correction in storage devices
Machine Learning
Feature selection, decision trees, cross-entropy loss
Cryptography
Measuring randomness, entropy of keys, information-theoretic security
Modern DevelopmentsDescription
Network Information Theory
Multi-user information theory, network coding
Quantum Information Theory
Quantum entanglement, quantum error correction
Computational Information Theory
Algorithmic information theory, Kolmogorov complexity
