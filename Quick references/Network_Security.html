Network Security - Quick Reference

ConceptDescription
Definition
Protection of computer networks from unauthorized access, misuse, malfunction, modification, destruction, or improper disclosure.
Goals
Confidentiality, Integrity, Availability (CIA Triad)
Layers
Physical, Data Link, Network, Transport, Application security
Security ThreatsDescription
Malware
Virus, worm, trojan, ransomware, spyware that damages systems
DoS/DDoS
Distributed Denial of Service attacks that overwhelm network resources
Man-in-the-Middle
Attacker intercepts and possibly alters communication between parties
Phishing
Fraudulent attempts to obtain sensitive information by disguising as trustworthy entities
Network Security ToolsDescription
Firewall
Network security system that monitors and controls incoming/outgoing traffic
IDS/IPS
Intrusion Detection/Prevention Systems that monitor network for malicious activity
VPN
Virtual Private Network for secure remote access to private networks
Antivirus
Software that prevents, detects, and removes malware
Authentication MethodsDescription
Single Factor
Password-only authentication
Two-Factor (2FA)
Two different authentication factors (something you know + have)
Multi-Factor (MFA)
Multiple authentication factors (something you know, have, are)
Biometric
Fingerprint, facial recognition, iris scan authentication
Encryption & ProtocolsDescription
SSL/TLS
Secure Sockets Layer/Transport Layer Security for encrypted web communication
IPSec
Internet Protocol Security for encrypted IP communication
WPA/WPA2/WPA3
WiFi security protocols for wireless network protection
SSH
Secure Shell for encrypted remote login and command execution
Network SegmentationDescription
VLAN
Virtual Local Area Networks to segment network traffic
DMZ
Demilitarized Zone for publicly accessible servers
Subnetting
Dividing network into smaller subnetworks for security and efficiency
Microsegmentation
Granular security policies at individual workload level
Security ProtocolsDescription
Kerberos
Network authentication protocol using secret-key cryptography
RADIUS
Remote Authentication Dial-In User Service for centralized authentication
TACACS+
Terminal Access Controller Access-Control System for network device access
802.1X
Network access control protocol for port-based authentication
Best PracticesDescription
Defense in Depth
Multiple layers of security controls throughout the network
Least Privilege
Grant minimum necessary access rights to users and systems
Regular Updates
Keep systems, software, and security patches current
Monitoring
Continuous network monitoring and logging for threat detection
