Digital Signal Processing - Quick Reference

ConceptDescription
Definition
Mathematical processing of discrete-time signals using digital computers or specialized digital hardware.
Applications
Audio processing, image processing, telecommunications, biomedical signal processing
Advantages
Flexibility, reproducibility, accuracy, ease of storage and transmission
Signal ClassificationDescription
Continuous vs Discrete
Continuous (analog) vs discrete (sampled) in time domain
Analog vs Digital
Analog (continuous amplitude) vs digital (quantized amplitude)
Deterministic vs Random
Predictable vs unpredictable signal behavior
Periodic vs Aperiodic
Signals that repeat vs non-repeating patterns
Sampling & QuantizationDescription
Sampling Theorem
Nyquist rate: sampling frequency must be at least twice the highest frequency component
Aliasing
Distortion when sampling rate is too low, higher frequencies appear as lower ones
Quantization
Converting continuous amplitude to discrete levels, introduces quantization noise
Resolution
Number of bits used to represent signal amplitude (8-bit, 16-bit, etc.)
Discrete TransformsDescription
DFT
Discrete Fourier Transform - converts time domain to frequency domain
FFT
Fast Fourier Transform - efficient algorithm for computing DFT
DTFT
Discrete-Time Fourier Transform - frequency representation of discrete signals
Z-Transform
Generalization of DTFT, useful for analyzing LTI systems
Filter TypesDescription
FIR
Finite Impulse Response - non-recursive, always stable, linear phase possible
IIR
Infinite Impulse Response - recursive, potentially unstable, more efficient
Low-pass
Allows low frequencies, attenuates high frequencies
High-pass
Allows high frequencies, attenuates low frequencies
Band-pass
Allows specific frequency range, attenuates others
Band-stop
Attenuates specific frequency range, allows others
Filter DesignDescription
Windowing
Method for FIR filter design using window functions (Hamming, Hanning, etc.)
Frequency Sampling
Design method based on desired frequency response samples
IIR Design
Butterworth, Chebyshev, Elliptic filter design from analog prototypes
Optimization
Parks-McClellan algorithm for optimal equiripple filter design
System PropertiesDescription
Linearity
System follows superposition principle
Time-Invariance
System behavior doesn't change with time
Causality
Output depends only on present and past inputs
Stability
Bounded input produces bounded output (BIBO)
ApplicationsDescription
Audio Processing
Noise reduction, equalization, audio compression
Image Processing
Filtering, enhancement, compression, feature extraction
Telecommunications
Modulation, demodulation, channel equalization
Biomedical
ECG, EEG, MRI signal processing and analysis
