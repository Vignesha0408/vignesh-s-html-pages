Microprocessors & Microcontrollers - Quick Reference

ConceptDescription
Microprocessor
CPU on a single integrated circuit. Requires external memory and I/O components.
Microcontroller
Single chip computer with CPU, memory, and I/O peripherals integrated.
Applications
Microprocessors: PCs, servers; Microcontrollers: embedded systems, IoT devices
Architecture ComponentsDescription
CPU
Central Processing Unit with ALU, control unit, and registers
ALU
Arithmetic Logic Unit for mathematical and logical operations
Control Unit
Manages instruction execution and system coordination
Registers
High-speed storage locations within CPU (PC, SP, IR, ACC)
Memory TypesDescription
RAM
Random Access Memory - volatile, used for temporary data storage
ROM
Read Only Memory - non-volatile, stores permanent data
Flash
Non-volatile memory that can be electrically erased and reprogrammed
Cache
High-speed memory between CPU and main memory for faster access
Instruction SetDescription
RISC
Reduced Instruction Set Computer - simple instructions, fixed length
CISC
Complex Instruction Set Computer - complex instructions, variable length
Instruction Cycle
Fetch, Decode, Execute, Store sequence
Addressing Modes
Immediate, Direct, Indirect, Register, Indexed addressing
Bus ArchitectureDescription
Data Bus
Carries data between CPU, memory, and I/O devices
Address Bus
Carries memory addresses from CPU to memory
Control Bus
Carries control signals for synchronization and coordination
Bus Width
Determines amount of data transferred simultaneously (8-bit, 16-bit, 32-bit, 64-bit)
Microcontroller PeripheralsDescription
GPIO
General Purpose Input/Output pins for digital I/O
ADC
Analog-to-Digital Converter for analog signal processing
DAC
Digital-to-Analog Converter for analog output generation
Timers/Counters
For time measurement, pulse generation, and event counting
Communication
UART, SPI, I2C, CAN for serial communication
Popular FamiliesDescription
Microprocessors
Intel x86, ARM Cortex-A, AMD, PowerPC
Microcontrollers
ARM Cortex-M, AVR, PIC, 8051, ESP32, STM32
Applications
General computing vs embedded control applications
Development ToolsDescription
IDE
Integrated Development Environment (Keil, MPLAB, Atmel Studio)
Compiler
Converts high-level code to machine code (GCC, IAR, Keil)
Debugger
Hardware/software tools for debugging (JTAG, SWD)
Simulator
Software tools to simulate microcontroller behavior
