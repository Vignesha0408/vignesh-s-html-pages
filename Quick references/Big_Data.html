Big Data - Quick Reference

ConceptDescription
Definition
Large, complex datasets that traditional data processing applications cannot handle. Characterized by volume, velocity, and variety.
The 5 V's
Volume, Velocity, Variety, Veracity, Value
Characteristics
Massive scale, diverse formats, fast generation, quality concerns, business value
Data TypesDescription
Structured
Data that fits in tables with defined schemas (databases, spreadsheets)
Semi-Structured
Data with some organizational properties but no fixed schema (JSON, XML)
Unstructured
Data without predefined structure (text, images, videos, social media)
Big Data TechnologiesDescription
Hadoop
Open-source framework for distributed storage and processing of large datasets
Spark
Fast, in-memory data processing engine for large-scale analytics
HDFS
Hadoop Distributed File System for storing large files across clusters
YARN
Resource management layer in Hadoop for job scheduling and cluster resources
NoSQL DatabasesDescription
Document Stores
MongoDB, Couchbase for JSON/BSON documents
Key-Value Stores
Redis, DynamoDB for simple key-value pairs
Columnar Databases
Cassandra, HBase for column-based storage
Graph Databases
Neo4j, Amazon Neptune for relationship-focused data
Data Processing ModelsDescription
Batch Processing
Processing large volumes of data at once (MapReduce, Spark Batch)
Stream Processing
Real-time processing of continuous data streams (Kafka, Storm, Flink)
Lambda Architecture
Combines batch and stream processing for real-time analytics
Analytics TypesDescription
Descriptive
What happened? Analyzes past data to understand historical patterns
Predictive
What might happen? Uses statistical models to predict future outcomes
Prescriptive
What should be done? Provides recommendations for optimal decisions
Machine Learning in Big DataDescription
Algorithms
Linear regression, clustering, neural networks, decision trees
Tools
MLlib (Spark), H2O, Mahout, TensorFlow on Spark
Applications
Customer segmentation, fraud detection, recommendation systems
ApplicationsDescription
Business Intelligence
Customer analytics, market analysis, operational optimization
Healthcare
Patient data analysis, drug discovery, personalized medicine
Finance
Fraud detection, algorithmic trading, risk management
IoT
Sensor data analysis, predictive maintenance, smart systems
